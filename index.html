Desktop Computer Components

Here are some terms you will see in relation to computer hardware.

Hard Disk Drives and Solid State Drives (devices) both store data permanently. SSDs are newer, and do not have the moving parts of their older equivalent - HDD. SSD are made from interconnected flash memory cells,similar to those in a USB drive (thumb drive or flash drive). SSDs are more reliable and more robust than the HDD equivalent.

The SSD (or HDD) stores data permanently on disk. Computer memory is just temporary storage, and the contents of RAM go away when the power goes.

RAM is regarded as primary storage because it is the immediate storage, whereas saving data to an SSD or a HDD is regarded as saving to secondary storage.

Basic Input Output System.

This is contained in a chip, inserted onto the motherboard by the motherboard manufacturer. The BIOS has instructions for loading the computer hardware. It includes POST (Power-On Self-Test) test, that verifies the computer can boot up. If the computer fails POST, a combination of beeps indicates the malfunction.

Form factor refers to the size, shape, and physical specifications of hardware or hardware components. The term applies to cases, motherboards, PSUs and more. Form factor is very important for determing connection compatibility.

A heat sink and fan (HSF) is an active cooling solution used to cool down integrated circuits in computer systems, commonly the central processing unit (CPU). A heat sink unit is composed of a metal alloy, formed into conductive fingers - this passive cooling unit draws heat away from the source, and the fan dissipates the heat.

Universal Serial Bus(USB) technology used to connect computers with peripheral devices. A USB port is a standardized way to connect a wide range of devices to a computer. Also referred to as Plug & Play.

A port is a connector with which an external device can be connected to the computer. Information flows between the connected device and the computer, through the port.

There are several ports designated wth numbers, that are programmatic docking points through which information flows from a program to the computer, or over the Internet.

CPU

Central Processing Unit

The brains of the computer.

Processors operate at high speed - today a typical speed might be upwards of 3 GigaHertz (GHz),3 billion instructions per second.

Motherboard

Main Circuit Board

This is the main circuit board inside the computer. All of the micro-component parts are attached to the motherboard. The Form Factor of the motherboard describes the dimensions, mounting holes, ports, processor suitability and more.

RAM

Random Access Memory

Temporary (volatile) storage in the computer, stores information for immediate use during processing. Also referred to as primary storage or main memory. Contents clear when there is no power source.

Case

Computer Case

Protective enclosure, also anchors the parts. May be referred to as computer chassis, tower, system unit, or cabinet.

The motherboard form factor should match the case.

Monitor

Monitor, Screen

Traditional output, the screen can be a touch-screen, which makes it an input device as well as an output device.

Screen size is measured diagonally, any upper vertex to the vertex diagonally opposite.

PSU

Power Supply Unit

The power supply converts from AC current to DC current, and regulates the DC output voltage to the lower tolerances required by the micro-components inside the case.

Choose a PSU form factor to match the case and motherboard.

Keyboard

Keyboard

The keyboard or keypad has become the default input device.

There are many styles to choose from, the most popular is the QWERTY keypad, named after the first 6 letters.

Expansion Card

Expansion Card

Adapters, or expansion cards, extend the functionality of the computer system by providing better quality graphics or audio, or by supplying networking capability.

Expansion cards are inserted into the expansion slots, or enabled with USB.

HDD

Hard Disk Drive

Permanent storage in the computer system.

This HDD records data magnetically and has multiple moving parts - spinning magnetic disks and a read-write head.

SSD

Solid State Drive/Device

Permanent storage in the computer system, no moving parts.

Data is stored on interconnected flash-memory chips that retain data even without power - like USB drives, but faster and more reliable. SSDs are typically smaller than HDDs.

OS

Operating System

An operating system (OS) is system software that manages computer hardware, software resources, and services for computer systems.

An OS enables better interactivity with the computer system. Examples inclide Linux, Windows, macOS.
